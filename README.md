# Exploratory-Testing-Online-Store

**Описание задания:**  
Проведено исследовательское тестирование корзины интернет-магазина с целью выявления дефектов в пользовательских сценариях и работе системы.

**Что было сделано:**  
- Выполнено тестирование функционала корзины и оплаты  
- Применены инструменты Chrome DevTools:
  - Console (ошибки в JavaScript)  
  - Network (проверка запросов и ответов)  
  - Elements (проверка DOM и UI)    
- Найдены и задокументированы дефекты
